# Pomodoro Project Manager

A full-stack project management application built with React, Spring Boot, and PostgreSQL.

## Tech Stack

### Frontend
- React
- Vite
- Axios
- Material UI

### Backend
- Java
- Spring Boot
- Spring Data JPA

### Database
- PostgreSQL

## Features

- Project management
- Session tracking
- Pomodoro timers
- Productivity analytics

In development
